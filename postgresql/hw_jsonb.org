* Database stats

Extract database usage statistics for all tables in single JSONB object
Structure of this object:

  #+BEGIN_SRC json
  {"<SCHEMA>": {
     "<TABLENAME>": {
            "rows": "<ROWS_ESTIMATE>",
            "index_size": "<TOTAL_INDEX_SIZE>",
            "table_size": "<TABLE_SIZE>",
            "index": {
                "<INDEXNAME>": {
                    "size": "<INDEX_SIZE>",
                    "type": "<INDEX_TYPE>"
                }
            }
     }
  }}
  #+END_SRC
